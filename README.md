## 📌 Project Overview

This project presents a descriptive epidemiologic analysis of reported Mpox (Monkeypox) cases in Florida from 2023 to 2024. The analysis was conducted to identify demographic trends, high-risk adult populations, and disparities in disease burden among Hispanic communities. Findings supported targeted public health outreach, surveillance planning, and health equity initiatives at the state level.

A condensed version of this analysis was presented as a poster at the International Conference on Aging in the Americas (ICAA 2025).

## 📊 Methods

### Analytic Approach

| Step                 | Description                                                                                                 |
| -------------------- | ----------------------------------------------------------------------------------------------------------- |
| Data Cleaning        | Standardized variables, created age categories, urban/rural classifications, and reporting-delay indicators |
| Descriptive Analysis | Case counts, demographic frequencies, and temporal trends                                                   |

| Timeliness Analysis  | Assessment of reporting delays and surveillance efficiency                                                  |
| Logistic Regression  | Bivariate and multivariable models evaluating hospitalization predictors                                    |

## 🔍 Key Findings

* Adults aged 25–44 years represented the highest proportion of reported Mpox cases.
* Hispanic/Latino populations experienced a disproportionate burden of disease relative to population share.
* Urban counties, particularly Miami-Dade, Broward, and Palm Beach, accounted for the majority of reported cases.
* Reporting timeliness varied across counties, highlighting opportunities to strengthen surveillance workflows.
* Multivariable logistic regression identified age group and Hispanic ethnicity as significant predictors of hospitalization after adjustment for sex and geographic factors.
